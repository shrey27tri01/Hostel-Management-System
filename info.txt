THIS IS A HOSTEL MANAGING SOFTWARE.
IT HELPS THE ADMIN TO KEEP TRACK OF HOSTEL PROPERLY.
IT ASKS FOR LOGIN CREDENTIALS SO THAT ONLY AUTHORISED PERSON CAN USE IT.
ADMIN CAN ADD THE STUDENTS IN HOSTEL IN THEIR RESPECTIVE BATCHES WITH AUTOMATIC ROLL-NUMBER AND ROOM-NUMBER ALLOCATION. 
ADMIN CAN DELETE STUDENTS BY THEIR BATCH AND ROLL NUMBER. 
ADMIN CAN SEE WHICH STUDENTS ARE INSIDE AND WHICH ARE NOT. 
ADMIN CAN SEACRH THE INFORMATION OF SOME STUDENTS BY HIS/HER NAME, BATCH, ROLLNUMBER. 
ADMIN CAN UPDATE SOME INFOERMATION ABOUT STUDENTS. WE CAN SEE THE INFORMATION OF WHOLE BATCH, GIRLS, BOYS ETC. 
IT HAS EXHAUSTIVE DATABASE OF STUDENTS COMPRISING DETAILS OF NEARLY 250 STUDENTS DISTRIBUTED IN 4 BATCHES.
